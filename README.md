Editor de Imagens Simples - Python CLI

Visão Geral do Projeto

Este é um projeto simples desenvolvido em Python que funciona como um editor de imagens básico via Linha de Comando (CLI). O objetivo é automatizar tarefas comuns de edição, como aplicação de filtros e redimensionamento, demonstrando o uso de bibliotecas de terceiros e manipulação de I/O de arquivos.

Tecnologias Utilizadas:
* Python 3
* Biblioteca Pillow (PIL - Python Imaging Library)

Funcionalidades Implementadas

O programa editor.py oferece as seguintes opções ao usuário:

1. Converter para Tons de Cinza (Grayscale): Transforma uma imagem colorida em preto e branco.
2. Redimensionamento: Permite ajustar a largura e altura da imagem para dimensões específicas.

Como Rodar o Projeto (Passo a Passo)

Siga estas instruções para clonar e executar o projeto no seu ambiente local.

1. Pré-requisitos
Certifique-se de ter o Python 3 e o gerenciador de pacotes pip instalados.

2. Instalação das Dependências
Este projeto requer a biblioteca Pillow. Instale-a usando o seguinte comando no seu terminal:
pip install Pillow

3. Execução
1. Coloque uma imagem (exemplo.jpg ou foto.png) na mesma pasta do arquivo editor.py.
2. Execute o script Python no seu terminal:
python editor.py
3. O programa iniciará e solicitará o Nome do Arquivo da imagem a ser editada. Após informar o nome, você poderá interagir com o menu para escolher a transformação desejada (1 para Grayscale, 2 para Redimensionar).
